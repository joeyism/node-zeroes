# zeroes
A simple tool to create an array with only 0's

## Installation
To install, run

    $ npm install --save zeroes

## Usage
To use, require it first by

    var zeroes = require("zeroes");

Then input the array length that you want

    zeroes() // []
    zeroes(0) // []
    zeroes(1) // [1]
    zeroes(5) // [1, 1, 1, 1, 1]
